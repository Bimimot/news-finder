About

This is diploma project was done at the Faculty of Web Development at Yandex.Practicum.

Project has resistance design and uses BEM metodology. Frontend bundle made with Webpack. There was configured style minification & JS transpilitation.

For searching news project uses API from newsapi.org. This REST API was developed for authorization and work with news cards. Backend was done with Node.js & MongoDB

Backend uses VPS with Nginx
