About

The project was developed at the Faculty of Web Development, Yandex School of Data Analysis.

Project has resistance design and uses BEM metodology. Frontend bundle was made with Webpack and includes style minification & JS transpilitation.

For getting news project uses newsapi.org API.

Project also has its own REST API for authorization and interaction with favorite cards. Backend based on Express.js and MongoDB.
